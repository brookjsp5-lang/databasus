## 一、功能概述

新建数据库向导是一个引导用户完成数据库配置和备份计划设置的多步骤表单界面。用户可以通过该向导快速配置数据库连接、备份类型、排程、存储位置、保留策略和通知设置。

## 五、API集成

### 5.1 使用的API端点

| API | 方法 | 功能 |
|-----|------|------|
| `/api/mysql-databases` | POST | 创建MySQL数据库 |
| `/api/postgresql-databases` | POST | 创建PostgreSQL数据库 |
| `/api/storages` | POST | 创建存储配置 |
| `/api/backup-configs` | POST | 创建备份配置 |

### 5.2 Cron表达式生成

向导自动将用户选择的排程转换为Cron表达式：

```typescript
// 按小时: 0 */N * * *
// 按日: M H * * *
// 按周: M H * * D
// 按月: M H D * *
// 自定义: 用户输入
```
